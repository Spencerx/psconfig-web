Seconds

The bin size for histogram calculations in terms of seconds. For example a value such as .001 means all histogram bins will be in milliseconds.

Packets Per Second

The mean average time between packets in seconds. For example, .1 means send 10 packets per second.

tests

The number of packets contained in each summary. This combined with packet_interval determines how often data is stored. For example, a packet_interval of .1 (10 packets per second) and sample_count of 600 stores a result every 60 seconds.

bytes

The size of the padding added to each packet in bytes (not including the IP/UDP headers)
